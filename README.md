# Q&A Service API
Интуитивно понятное API для создания и управления вопросами и связанными с ними ответами. Сервис обеспечивает целостность данных: при удалении основного вопроса все ассоциированные с ним ответы автоматически каскадно удаляются, избавляя вас от необходимости ручной очистки. Идеально подходит для реализации разделов FAQ или систем поддержки.

## Запуск Docker compose
Для начала убедитесь, что ваш локальный Docker Server запущен.

Скачав репозиторий с git, откройте консоль, перейдите в папку проекта и выполните команду:
```cmd
docker compose up
```
Дождитесь запуска всех контейнеров.

### В Docker compose находятся 2 контейнера:

1. База данных Postgres на порту `6432` (Для подключения к базе локально)
2. Само приложение FastApi


Перейдя по [ссылке](http://localhost:8000/docs#/) вы попадете на страницу документации для изучения API.
